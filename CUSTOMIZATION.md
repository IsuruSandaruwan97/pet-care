# Customization Guide

This template is designed so the most common buyer edits are grouped in a few predictable places.

## 1. Brand, Contact, SEO, and Social Links

Edit `src/config/site.ts` for:

- clinic name and legal name
- tagline and description
- production site URL
- address
- phone, emergency phone, and email
- business hours
- social links
- default SEO title, description, and Open Graph image

Set `NEXT_PUBLIC_SITE_URL` in `.env` before deploying so sitemap, robots, Open Graph, and structured data use the correct domain.

## 2. Page Content

Most reusable page content lives in `src/data/index.ts`, including:

- trust chips
- stats
- services
- facilities
- team members
- pricing
- testimonials
- pet care tips
- FAQs
- appointment reason options

Navigation labels and section targets live in `src/constants/index.ts`.

## 3. Images and Videos

Assets are stored in `media/images` and `media/videos`.

All included demo images and videos were generated by the template author using Kling AI. Buyers can keep them as demo media where permitted by the template license and applicable Kling AI terms, or replace them with their own clinic assets.

To add a new asset:

1. Place the file in the correct `media` folder.
2. Add a key to `src/app/api/media/[asset]/route.ts`.
3. Reference it as `/api/media/your-key`.

The media API supports HTTP range requests for MP4 files.

## 4. Appointment Form

The appointment form posts to `/api/appointments`.

By default, it runs in demo mode and returns a success response. To forward submissions to a real service, set:

```env
APPOINTMENT_WEBHOOK_URL="https://your-webhook-url"
```

This works with tools such as Formspree, Zapier, Make, n8n, or a custom API endpoint.

## 5. Google Map

Set `NEXT_PUBLIC_MAP_URL` in `.env`.

If this value is empty, the template shows a placeholder map area.

## 6. Colors and Fonts

Theme tokens live in `tailwind.config.ts`.

Global styles live in `src/app/globals.css`, while page-specific CSS lives in `src/screens/*/styles.css`.

Fonts are stored in `public/assets/fonts`.

## 7. Optional Effects

Right-click protection is disabled by default. Enable it with:

```env
NEXT_PUBLIC_RIGHT_CLICK_GUARD="true"
```

The custom cursor, preloader, scroll progress, reveal animations, and scroll-to-top button are wired in `src/app/layout.tsx` and page components.
